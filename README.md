# MorphStrata

Layer-specific perturbation strategy for generating diverse student models in Moving Target Defense (MTD) for Transformer-based time-series forecasting.

MorphStrata extends the [Morphence](https://arxiv.org/abs/2108.13952) framework by confining stochastic Gaussian noise to distinct Transformer components (self-attention projections, feed-forward networks, layer normalization, input/output projections) rather than perturbing all parameters uniformly. This produces a student pool with greater structural heterogeneity and reduced adversarial transferability across ensemble members.

---

## Method

Given a trained base Transformer with parameters `θ`, each student is generated by:

```
θ_i = θ + m_k ⊙ η_i,    η_i ~ N(0, σ²I)
```

where `m_k` is a binary mask selecting stratum `S_k`. Each student is perturbed in a different functional region of the computational graph. The Vanilla (Morphence) baseline applies the same noise globally with no masking.

All students are adversarially fine-tuned after generation. At inference, `M` students are sampled uniformly and their predictions averaged.

### Strata

| Stratum | Parameters targeted |
|---|---|
| Encoder self-attention | Q, K, V, output projections |
| Decoder attention (seq2seq only) | Self-attention + cross-attention projections |
| Feed-forward network (FFN) | `linear1`, `linear2` in encoder and decoder |
| Layer normalization | `gamma`, `beta` across all norm layers |
| Input projection + head | Input embedding projection, output regression head |

## Results Summary

Evaluated against FGSM, BIM, and PGD (white-box, L-inf) across ε ∈ {0.1, 0.2, 0.3, 0.5}, 30 randomized trials per configuration.

- **AEP:** MorphStrata achieves the lowest adversarial RMSE across all attacks and perturbation budgets. Under BIM at ε=0.5, the undefended base model reaches RMSE 3.865; MorphStrata holds at 0.079 (97.97% reduction).
- **ECL:** MorphStrata and the Vanilla Ensemble are competitive; MorphStrata leads under PGD.
- **JENA:** Vanilla Ensemble leads under FGSM and PGD, consistent with the dataset's strong long-range periodicity. Both ensembles are near-identical under BIM.
- **Overhead:** Layer targeting adds less than 1.1% wall-clock training time over the Vanilla Ensemble across all nine dataset-attack conditions.
- **Weight diversity:** On AEP, MorphStrata produces larger pairwise L2 distance than Vanilla across all three attacks (+34.2% under FGSM, +17.9% under BIM, +9.0% under PGD), and is the only dataset where it achieves the lowest RMSE in every condition. This supports the positive correlation between weight diversity and defense effectiveness.

## Repository Structure

```
morphstrata/
└── notebooks/
    ├── JENA.ipynb
    ├── ECL.ipynb
    └── AEP.ipynb
```

Each notebook is a self-contained pipeline covering:

- Data preprocessing and chronological splitting
- Base Transformer training
- Vanilla (Morphence) and MorphStrata student generation
- Adversarial fine-tuning under FGSM, BIM, and PGD
- 30-run Monte Carlo evaluation
- Pairwise L2 weight diversity and differential immunity metrics
- Per-stage wall-clock, VRAM, and CPU RAM monitoring
